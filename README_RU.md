## 🎮 Реализованные Механики

### Основной Геймплей
- 8 типов растений × 3 уровня редкости = 24 варианта
- Рост в реальном времени (1% в минуту)
- Система здоровья с деградацией
- Расчет красоты на основе роста и здоровья
- 3 действия: полив 💧, удобрение 🌱, подрезка ✂️
- Продажа с динамическим ценообразованием

### Экономика
- Монеты как основная валюта
- Покупка семян (10/50/200 монет)
- Рыночные тренды каждые 2 минуты (+50% / -20%)
- Система уровней (каждые 1000 монет продаж)

### Автоматизация
- Садовник (автополив каждый час) - 500 монет
- Маркетолог (пассивные продажи по 70% цены) - 1000 монет

### Расширения
- Оранжерея (+2 полки) - 800 монет
- Доставка (пассивный доход каждые 3 минуты) - 1500 монет

### Сохранения
- Автосохранение в UserDefaults
- Экспорт сохранения в JSON
- Импорт сохранения из JSON
- Расчет оффлайн прогресса

## 🐛 Исправленные Ошибки

### ✅ JSON Parsing Error - ИСПРАВЛЕНО
**Проблема:** Словарь `[PlantType: [PlantRarity: Int]]` не может быть закодирован в JSON, так как Swift не поддерживает enum-ключи в JSON по умолчанию.

**Решение:** Создана кастомная реализация Codable в `SeedInventory.swift`:
- Внутреннее хранилище использует `[String: [String: Int]]`
- Computed property `seeds` конвертирует в/из `[PlantType: [PlantRarity: Int]]`
- Полная совместимость с JSON сериализацией
- Сохранена типобезопасность API

## 🚀 Как Запустить

### Шаг 1: Создать Xcode Проект
1. Открыть Xcode
2. File → New → Project
3. iOS → App
4. Имя: `PlantTycoon`
5. Interface: **SwiftUI**
6. Language: **Swift**
7. Minimum iOS: **16.0**

### Шаг 2: Добавить Файлы
1. Удалить дефолтные ContentView.swift и PlantTycoonApp.swift
2. Добавить всю папку `PlantTycoon/` в проект
3. Убедиться, что "Copy items if needed" включено

### Шаг 3: Запустить
1. Выбрать симулятор (iPhone 15)
2. Нажать ⌘ + R
3. Игра запустится! 🎉

**Подробная инструкция:** см. `XCODE_SETUP.md`

## 📱 Первый Запуск
