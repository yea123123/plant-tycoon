🐛 BUG FIXES APPLIED
─────────────────────────────────────────────────────────────────
❌ ORIGINAL ERROR: "API Error: Failed to parse JSON"

🔍 ROOT CAUSE:
   SeedInventory used [PlantType: [PlantRarity: Int]] dictionary
   Swift cannot encode enum-keyed dictionaries to JSON by default

✅ SOLUTION IMPLEMENTED:
   • Created custom Codable implementation in SeedInventory.swift
   • Internal storage: [String: [String: Int]] (JSON-compatible)
   • Public API: [PlantType: [PlantRarity: Int]] (type-safe)
   • Computed property converts between formats
   • Full JSON serialization support
   • Maintains type safety and API compatibility

📝 FILES MODIFIED:
   PlantTycoon/Models/SeedInventory.swift (lines 10-85)

✅ RESULT: Full save/load functionality working correctly
